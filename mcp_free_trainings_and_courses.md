## Free MCP Fundamentals Courses

### Beginner


| Course / resource title & provider                                                                                                                                                    | Summary                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**Official MCP Documentation: Getting Started**](https://modelcontextprotocol.io/docs/getting-started/intro) - Model Context Protocol project                                        | This is the protocol maintainers’ introduction to MCP: what problem it solves, how hosts, clients, and servers fit together, and how tools, resources, and prompts expose capabilities to models. It is the best place to learn canonical terms and mental models before reading tutorials or building integrations.                                                                               |
| [**Build an MCP Server**](https://modelcontextprotocol.io/docs/develop/build-server) - Model Context Protocol project                                                                   | A guided tutorial that walks you through implementing an MCP server (using a weather-style example), defining tools, wiring Claude Desktop as a host, and debugging common issues. It is aimed at developers who want their first concrete server-side MCP project rather than theory alone.                                                                                                              |
| [**Build an MCP Client**](https://modelcontextprotocol.io/docs/develop/build-client) - Model Context Protocol project                                                                 | Explains how to build an LLM-powered chat client that connects to MCP servers so the model can call tools and use shared context. It pairs naturally with the server tutorial once you understand how servers expose capabilities.                                                                                                                                                                     |
| [**MCP Quickstart Resources**](https://github.com/modelcontextprotocol/quickstart-resources) - Model Context Protocol project                                                         | A minimal reference repository that backs the official server and client quickstarts with copy-pasteable starter code. The examples are intentionally small; treat them as learning aids and harden authentication, validation, and deployment before using patterns in production systems.                                                                                                               |
| [**Model Context Protocol Course**](https://huggingface.co/learn/mcp-course/unit0/introduction) - Hugging Face                                                                        | A structured, multi-unit Hugging Face course covering MCP fundamentals, architecture, common SDKs and frameworks, assignments, and optional community projects. It offers a clearer progression than scattered blog posts and includes a certificate-oriented track for learners who want checkpoints.                                                                                               |
| [**MCP for Beginners**](https://github.com/microsoft/mcp-for-beginners) - Microsoft                                                                                                   | Microsoft’s open-source curriculum spans MCP basics through security, transports, testing, deployment, authentication, Azure scenarios, and case studies across C#, Java, JavaScript, Python, TypeScript, Rust, and adjacent tooling. It targets teams that want opinionated modules and examples rather than a single-language-only tutorial set.                                                  |
| [**Model Context for Beginners**](https://techcommunity.microsoft.com/blog/azuredevcommunityblog/model-context-for-beginners/4414964) - Microsoft Developer Community Blog            | A Tech Community article that orients readers to Microsoft’s broader MCP learning path, summarizes how the GitHub curriculum fits together, and touches security basics at a high level. Useful as a short roadmap before cloning the full beginner repo.                                                                                                                                               |
| [**Building Agents with Model Context Protocol - Full Workshop**](https://www.classcentral.com/course/youtube-building-agents-with-model-context-protocol-full-workshop-with-mahesh-murag-of-anthropic-457629) - AI Engineer / YouTube | A full-length recorded workshop that explains MCP in the context of building agents, discusses practical integration themes, and outlines ecosystem direction. Because it is a talk-style recording, pair it with the official specification and tutorials when you need exact API behavior or version-specific details.                                                                                   |

## Free MCP Security Trainings

### Beginner / foundational (security literacy and reference)


| Course / resource title & provider                                                                                                                                                          | Format                           | Summary                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**MCP Security Best Practices**](https://modelcontextprotocol.io/docs/tutorials/security/security_best_practices) - Model Context Protocol project                                           | Official security documentation  | Official guidance from the MCP project on prominent abuse patterns such as confused deputy problems, token passthrough mistakes, SSRF via tools, session hijacking surfaces, and OAuth-related pitfalls, with suggested mitigations and secure design habits. It should be a primary reference for anyone implementing hosts, servers, or auth bridges.                                                                                                |
| [**MCP Security Cheat Sheet**](https://cheatsheetseries.owasp.org/cheatsheets/MCP_Security_Cheat_Sheet.html) - OWASP Cheat Sheet Series                                                      | Cheat sheet / practitioner guide | The OWASP cheat sheet distills deployment hygiene (least privilege, transport protection, isolation between servers), prompt-injection and tool-poisoning considerations, and operational checklist items defenders expect in reviews. It works well as a quick refresher for engineers and as a discussion aid between security and platform teams.                                                                                                      |
| [**Why MCP Enabled Agents Are the Next Cyber Battleground**](https://genai.owasp.org/event/why-mcp-enabled-agents-are-the-next-cyber-battleground/) - OWASP GenAI Security Project            | Webinar                          | An OWASP GenAI webinar framing how MCP expands the attack surface for agent systems, including themes like prompt injection, memory-oriented abuse, and orchestration risks across multiple tools and identities. It is oriented toward awareness for both executives and hands-on practitioners; recording availability may depend on how the project publishes past sessions.                                                                          |
| [**Building Secure MCP Servers**](https://snyk.io/articles/building-secure-mcp-servers/) - Snyk                                                                                             | Article / developer guide        | A Snyk-authored article that translates familiar web and API security classes-command injection, path traversal, SQL injection-into MCP server code paths and configuration choices, reinforcing defense-in-depth when tools wrap shells, databases, or filesystem access. It complements protocol-level docs with concrete secure-coding guidance.                                                                                                       |

It bridges high-level threat discussion into engineering decisions you can defend in design reviews.                                                                                                                                 |
| [**Secure AI Development Learning Path**](https://learn.snyk.io/learning-paths/secure-ai-development/) — Snyk Learn                                                                      | Learning path              | Snyk Learn’s broader secure-AI path covers resilient agent applications and explicitly includes MCP-related material alongside adjacent topics. Progress tracking and badges help learners who want modular lessons rather than a single long PDF; expect MCP coverage as part of a wider curriculum.                                                                                                                                                                    |
| [**MCP Security Workshop**](https://www.apisecuniversity.com/resources/mcp-security-workshop) — APIsec University                                                                          | Workshop                   | APIsec University promotes a workshop that connects MCP security themes with classic API security concerns—abuse cases, auth models, and secure consumption patterns. Exact agenda, duration, and delivery format can change; confirm the current offering on the vendor page before planning attendance or stakeholder briefings.                                                                                                                                          |
| [**MCP Security Summit Workshop: Sherpa**](https://github.com/Azure-Samples/sherpa) — Azure Samples / Microsoft                                                                          | Guided security lab        | Sherpa is a Microsoft/Azure Samples guided lab that walks through enterprise-oriented controls for MCP servers: OAuth flows, managed identities, Key Vault usage, gateways and private endpoints, content safety, validation, PII handling, logging, monitoring, threat-detection hooks, and validation exercises framed from red and blue perspectives. Expect Azure-centric assumptions and optional cloud spend for hands-on environments.                         |
| [**mcp-security-workshop**](https://github.com/antoninBr/mcp-security-workshop) — Community GitHub project                                                                               | Hands-on security workshop | A community workshop repository demonstrating offensive security scenarios such as side-channel execution paths, secret exfiltration patterns, supply-chain and container poisoning ideas, prompt injection chains, and auditing angles specific to MCP deployments. Run artifacts only in disposable lab VMs or sandboxes; never point exercises at production endpoints or real credentials.                                                                         |


### Advanced / hands-on attack and vulnerable labs


| Course / resource title & provider                                                                                                                                                          | Format                          | Summary                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**Damn Vulnerable Model Context Protocol (DVMCP)**](https://www.augmentcode.com/mcp/damn-vulnerable-model-context-protocol-dvmCP) — Augment Code / community                               | Vulnerable lab                  | DVMCP is an intentionally vulnerable MCP-style lab environment where you can practice exploiting and defending realistic weaknesses: prompt injection, tool poisoning, excessive permissions, rug-pull patterns, tool shadowing, indirect injection, token theft, and malicious execution paths. Treat it strictly as education or controlled red-team evidence generation; label artifacts clearly and isolate networks because PoC code can be hazardous if reused carelessly.                                                                                             |
